A propos

Les Google Developer Groups (GDGs) sont des groupes de développeurs intéressés par les technologies Google : GWT, Android, App Engine, les APIs Google, mais également les techno du Web comme HTML5, CSS3... Nous nous intéressons à tout type de techno que ce soit dans le domaine de la mobilité, du cloud ou du web.

Le GDG Nantes


L’objectif du GDG Nantes est d’organiser des évènements de manière régulière :
  • Conférences sur les technologies Google, Web ...
  • Des apéros ;) autour de sujets centraux afin de se rencontrer et boire un verre ensemble
  • Des concours (Hackathon...)

Le GDG Nantes est également partenaire d’évènements nationaux ou d'autres communautés tels que les JUG.
